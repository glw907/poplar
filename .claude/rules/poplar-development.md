description: Development workflow for poplar implementation passes
---

When the user says "continue development", "next pass", "start the next pass", or "continue" in the context of poplar work:
1. Read `docs/poplar/STATUS.md` to find the current pass and starter prompt
2. Read `docs/superpowers/specs/2026-04-09-poplar-design.md` for the full spec
3. Read `docs/poplar/architecture.md` for accumulated design decisions
4. Execute the pass described in the starter prompt
5. At the end of the pass, follow the pass-end checklist in STATUS.md
