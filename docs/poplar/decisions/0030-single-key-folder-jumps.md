status: accepted

## Context

Bubbletea sends one KeyMsg per keypress. Multi-key
sequences require a state machine. Uppercase avoids conflict with
lowercase triage keys (d/a/s).

## Decision

Uppercase single keys (I/D/S/A/X/T) jump to
canonical folders from any context. Not multi-key sequences.

## Consequences

No follow-on notes recorded.
