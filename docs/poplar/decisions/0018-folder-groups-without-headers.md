title: Folder groups without headers
status: accepted
date: 2026-04-10
---

## Context

The grouping is self-evident from folder names and
icons. Headers would label the obvious and add visual noise. Groups
exist in the data model for sort order and folder jump targeting
(via command mode).

## Decision

Three folder groups (Primary, Disposal, Custom)
separated by blank lines in the sidebar. No rendered group headers.

## Consequences

No follow-on notes recorded.
